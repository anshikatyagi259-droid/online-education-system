# Online Education System

A modern and simple Online Education System developed using HTML, CSS, and JavaScript.  
This project provides students with an interactive learning platform including login system, courses, quiz section, dark mode, and student dashboard.

---

## 📌 Features

- 🔐 User Registration & Login
- 🏠 Student Dashboard
- 📚 Courses Section
- 📝 Quiz System
- 🌙 Dark Mode / Light Mode
- 👨‍🎓 Student Profile
- 🎨 Responsive and Attractive UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
